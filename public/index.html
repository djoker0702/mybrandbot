About us

This application is a messenger chatbot used to help the users to manage their favorite brands online websites and keep them tunned to the latest seasonal sales and discounts ...

It's developped using nodejs, express and mongoDb for the backend, api.ai for the frontend and the standar conversation flow

Private policy

This application will have full access to some user's profile information such as profile pic, full name, timezone, country, gender....

The chatbot will ask the user about his favorite brands names, websites ..., and it will start follow them and keep the user updated about the latest promotions and discounts
